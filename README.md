\# Project Template



\## Overview

This is a template for empirical research projects combining R, Python, and LaTeX.



\## Structure

\- `data/`: All data files (raw, intermediate, processed)

\- `code/`: Analysis scripts and functions

\- `output/`: Generated figures, tables, and models

\- `paper/`: LaTeX manuscript

\- `presentations/`: Conference slides



\## Setup Instructions

1\. Install R and RStudio

2\. Install Python

3\. Install LaTeX (MiKTeX or TeX Live)

4\. Run `renv::restore()` to install R packages

5\. Run `pip install -r requirements.txt` to install Python packages



\## Usage

Run the master script: `source("run\_all.R")`
