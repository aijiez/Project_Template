\# Data Documentation



\## Directory Structure



\### 0\_raw/

Original, immutable data files. \*\*NEVER modify files in this directory.\*\*



\### 1\_selected\_raw/

Partially processed data at various cleaning stages. Delete useless variables. Save space. 



\### 2\_processed\_data/

Final analysis-ready datasets.



\## Confidentiality

Register data is confidential and subject to data agreements. 

Do not commit, share, or publish sensitive data.
